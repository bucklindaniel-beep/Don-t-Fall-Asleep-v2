# Next Smoke Test Steps

## Current Objective

Run the first real upload-mode transcript smoke test using one full `.md` transcript generated by the PowerShell intake script.

## Required Input

One full `.md` transcript uploaded directly into Claude.

## Do Not Use Yet

- Full production pipeline
- Multiple transcript batch
- Image generation
- ElevenLabs generation
- Local repo write automation

## Success Criteria

- Claude detects uploaded `.md` file
- Transcript depth gate passes
- Structure extraction runs
- Narration mechanics extraction runs or gives a valid block reason
- Pattern distillation separates structure from narration
- Indexed preview is generated but not written
- No memory updates occur
