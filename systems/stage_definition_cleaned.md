# Stage Definition — CLEANED

## Purpose

Prepare transcript text for analysis by removing noise only.

---

## ALLOWED ACTIONS

- fix transcription errors
- remove timestamps
- normalize spacing
- remove formatting artifacts

---

## PROHIBITED ACTIONS

- paraphrasing
- summarizing
- rewriting
- perspective shifting
- clarity rewriting
- narrative framing

---

## CRITICAL RULE

Original phrasing MUST be preserved.

If meaning, tone, or perspective changes → stage fails.

---

## SUCCESS CRITERIA

- original meaning preserved
- original wording preserved
- no added interpretation
