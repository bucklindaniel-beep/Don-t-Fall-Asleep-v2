# Image Prompt Pattern Extraction

## Purpose
Convert image prompt references into reusable visual prompting mechanics.

## Pattern Fields
Each extracted pattern should include:
- pattern name
- source type
- visual function
- when to use
- when not to use
- prompt mechanic
- continuity risk
- density cost

## Scoring Considerations
Prefer patterns that improve:
- visual consistency
- renderability
- emotional clarity
- shot-to-shot continuity
- prompt efficiency

Deprioritize patterns that cause:
- over-description
- style drift
- conflicting camera language
- excessive token cost
- generic cinematic filler

## Output Rule
Patterns must be rewritten as original system mechanics. Never preserve source phrasing.
