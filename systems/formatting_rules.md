# Formatting Rules

## Purpose

Keep system files clean, parseable, and execution-safe.

---

## Rule

Do not use emojis or decorative symbols in:

- /systems/
- /prompts/
- project instructions
- execution files
- repository control files

---

## Allowed Exceptions

Emojis and decorative symbols are allowed only in:

- YouTube packaging outputs
- creative content layers
- user-facing marketing copy when explicitly requested

---

## Enforcement

If emojis or decorative symbols appear in system-control files:

- remove them immediately
- treat them as formatting violations

Formatting should prioritize:

- clarity
- consistency
- token efficiency
- machine readability
