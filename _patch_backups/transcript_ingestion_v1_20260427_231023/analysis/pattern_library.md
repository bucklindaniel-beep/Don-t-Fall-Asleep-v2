## P-024 — Impossible Knowledge Confirmation

### Type
Core Pattern

### Definition
A source with no plausible access to specific information produces verifiable details that are independently confirmed as accurate, eliminating all reasonable explanations and forcing acceptance of an impossible or supernatural origin.

### Mechanic
A claim containing specific, checkable details is provided by a source that cannot logically possess the information. External verification confirms accuracy, collapsing all rational alternatives.

### Common Uses
- Past life memory claims
- Unexplained historical recall
- Unknown identity recognition with verification
- Predictive or hidden knowledge revealed and confirmed

### Structural Notes
- Resolves dissonance into certainty (unlike P-015)
- Requires independent verification step
- Must eliminate plausible information pathways
