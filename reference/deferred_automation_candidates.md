# Reference — Deferred Automation Candidates

STATUS: FUTURE / INACTIVE

## Purpose
Preserve useful automation candidates without introducing active systems before validation is complete.

## Candidates
- Make: future orchestration layer after pipeline and quality lock
- Zapier: lightweight notification/trigger layer if needed
- Postman: API testing for OpenAI / ElevenLabs / YouTube endpoints
- Claude connectors: repository access, git/diff review, controlled filesystem scope
- PowerShell bridge: deterministic local write/run layer

## Boundary
These tools are execution glue only. They must not replace Claude stage logic, pattern governance, or repository authority.
