# System Improvement Roadmap — Next Steps for Repository Quality

## Source Context

This roadmap was created after the first pipeline test using **"The Guest."**

### Main Finding
The structural pipeline is working.

What is already strong:
- escalation structure
- scene breakdown
- shotlist logic
- visual prompt continuity
- camera / realism systems
- overall pipeline architecture

### Main Bottleneck
The **narration voice layer** is underdeveloped.

The writing is:
- structurally correct
- visually coherent
- technically compliant

But it still tends to sound:
- clinical
- emotionally flat
- too written for the page
- insufficiently optimized for spoken AI narration
