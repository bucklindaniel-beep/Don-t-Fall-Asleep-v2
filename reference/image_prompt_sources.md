# Image Prompt Sources

This file tracks approved reference source types for image prompt analysis.

## Allowed Source Types
- user-provided prompt examples
- public prompt galleries
- successful internal prompts
- generated image review notes

## Rules
- Sources are for analysis only.
- Do not copy prompt text into system prompts.
- Do not imitate named living artists or proprietary styles.
- Extract reusable mechanics and rewrite them in original language.

## Candidate Source Notes
- PromptHero and similar galleries may be useful for studying prompt structure, density, and visual control patterns.
