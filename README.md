# Horror Knowledge Base

A modular knowledge system for building high-retention horror stories, narration scripts, scene systems, and AI-assisted creative workflows.

## Purpose

This repository is designed to help transform raw source material into reusable horror writing knowledge.

It is organized to support:
- transcript ingestion and processing
- pattern extraction
- style analysis
- structured horror writing systems
- reusable prompts and playbooks

## Core Workflow

The repository follows this general flow:

`transcripts -> analysis -> frameworks -> playbooks -> prompts`

### Meaning of each layer

- **transcripts**: source material and transcript processing pipeline
- **analysis**: extracted techniques, patterns, and style observations
- **frameworks**: reusable systems and models derived from analysis
- **playbooks**: practical execution guides for building stories and scenes
- **prompts**: AI-ready instructions built on top of the system
- **wordbanks**: categorized language banks for horror writing
- **reference**: foundational writing knowledge and external principles
- **templates**: reusable file templates for consistency
- **maps**: high-level relationship and flow documents
