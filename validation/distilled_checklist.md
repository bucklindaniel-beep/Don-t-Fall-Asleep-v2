# DISTILLED Checklist

## PASS
- Pattern names are literal and observable
- Evidence is directly grounded
- No cross-story contamination
- No inferred causality
- No vague insight language
- No framework creation
- Each story processed independently

## FAIL
- Pattern names are abstract
- Cause/effect is invented
- Story details leak into another story
- Output sounds like analysis commentary
- Pattern promotion language appears
