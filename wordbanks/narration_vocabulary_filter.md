# Wordbank — Narration Vocabulary Filter

## Purpose

Filters out written, clinical, or system-like language that sounds unnatural when spoken aloud.

Applies to:
- narration scripts
- first-person storytelling
- spoken AI voice output

Does NOT apply to:
- image prompts
- shotlists
- editing scripts
- visual systems

---

## Core Rule

If the narrator would not naturally say it in conversation, remove it.

---

## Replacement Map

| Avoid | Prefer |
|---|---|
| the lock lever | the lock |
| horizontal position | sideways / not locked |
| object displacement | something moved |
| repositioned | moved |
| misaligned | off / not right / shifted |
| inconsistency | something was off |
| anomaly | something wrong / something strange |
| unease | that feeling / something in my gut |
| dread | a weight / something heavy |
| recognition | I realized / it hit me |
| proximity increased | it was closer / getting closer |
| frequency increased | it kept happening |
| environmental change | something changed / the room felt different |
| spatial arrangement | never use |
| balanced composition | never use |

---

## Speech Preference Rules

Narration should prefer:

- owned language
- casual phrasing
- direct object naming
- simple sentence-level observation

Examples:

### Bad
There was a subtle environmental inconsistency in the room.

### Better
Something in the room was off.

### Bad
Object displacement suggested outside intervention.

### Better
Something had moved. I knew I hadn't touched it.

---

## Failure Conditions

Narration fails if:

- language sounds academic
- vocabulary sounds like it came from a framework
- the narrator describes reality like a camera log
