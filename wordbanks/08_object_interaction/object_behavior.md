# Wordbank — Object Interaction

## Purpose
Controls environmental anomalies through objects.

---

## Entries

### Subtle (Pattern Formation)
- slightly out of place [stage: early]
- turned at a different angle [stage: early]
- not where it was before [stage: early]

### Confirmed (Escalation)
- moved without explanation [stage: mid]
- left open when it wasn’t [stage: mid]
- returned to the same position [stage: mid]

### Aggressive (Intrusion)
- positioned deliberately [stage: late]
- aligned too perfectly [stage: late]
- shifted while observed [stage: late]

---

## Pairing Rules
- pair with spatial awareness for clarity
- reinforce patterns before escalating

---

## Anti-Repetition
- escalate object behavior; do not reuse the same anomaly
