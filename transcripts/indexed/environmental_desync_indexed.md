# Pattern Index Entry

## Pattern Name
Environmental Desynchronization

---

## Description
A horror mechanism where physical spaces subtly change independent of the subject’s interaction, creating distrust between memory, evidence, and reality.

---

## Use Cases
- Apartment-based horror
- Isolation settings
- Psychological horror narratives

---

## Key Signals
- Objects moving slightly
- Repeated inconsistencies
- Contradictory evidence (photos, recordings)
- Quiet, deliberate movement

---

## Escalation Path
1. Minor unnoticed changes
2. Confirmed inconsistency
3. Evidence contradiction
4. Passive observation phase
5. Live confirmation event

---

## Integration Notes
- Pair with:
  - unseen presence entities
  - spatial manipulation systems
- Avoid:
  - early visual reveal
  - excessive explanation
