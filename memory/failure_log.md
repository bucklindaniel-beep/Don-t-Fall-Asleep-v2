## High-Risk Failure Types
- Flat setup
- Repetitive narration rhythm
- Static camera language
- Broken continuity
- Weak escalation
- Overwritten image prompts
- Clinical vocabulary
- Spoken-delivery failure
- Redundant scenes
- Low visual variation

## Escalation Rule
If a failure repeats more than twice, convert its prevention rule into a hard project rule.
